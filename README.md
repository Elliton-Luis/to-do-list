# To-Do List

Aplicação simples para gerenciar uma lista de tarefas, desenvolvida com Laravel, Livewire e Tailwind CSS.

---

### Funcionalidades

Criar, listar, editar e excluir tarefas.

Exclusão lógica (soft delete) e restauração.

Filtro por status (pendente ou concluída).

Validação de dados no backend.

Rotas RESTful com Route::resource().

Componentes dinâmicos com **Livewire**.

Layout responsivo com **Tailwind CSS.**

(Opcional) Autenticação para proteger as rotas.

### Tecnologias

PHP 8.2+

Laravel 11

Livewire

Tailwind CSS

MySQL
